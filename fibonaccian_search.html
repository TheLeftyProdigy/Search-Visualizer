FIBONACCIAN SEARCH

INTRODUCTION

1. The working of Fibonacci Search includes dividing the array into unequal parts.
2. We first find the smallest fibonacci number (m) that is greater than or equal to the length of the array.
3. Next we search for the element at index i = (m-2).
4. If the number to be searched is greater than the number at the index then we recursively search the latter part of the array else, we recursively search the earlier part of array.

Fibonacci Search actually divides the given array in unequal parts.
